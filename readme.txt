Show your visitors how to get to you, during their journey.

== Description ==

Enroute, shows your users how to get from their destination to yours. By the use of Google Maps API, it will calculate the shortest route from A to B and tell you exactly which roads to take.

Want to contribute? [Fork the GitHub repository](https://github.com/iammathews/enroute "Fork the GitHub repository"). For support please visit the [Support forum](http://wordpress.org/support/plugin/enroute "Support forum").

== Installation ==

1. Upload the folder `enroute` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in your WordPress dashboard

= How do I get the route to show? =

Edit the page you would like to add Enroute to, right above the content area there is a "Add Enroute" button, which will generate a shortcode based on the settings you choose.
